## 1. 인증 테스트

### AUTH-001: 정상 로그인

| 항목 | 내용 |
|------|------|
| **시나리오 ID** | AUTH-001 |
| **시나리오 설명** | 유효한 자격 증명으로 로그인 시 정상적으로 인증되는지 확인 |
| **사전 조건** | - 활성 상태의 회원 계정이 존재<br>- 사용자가 로그아웃 상태 |
| **테스트 단계** | 1. `/auth/login` 페이지로 이동<br>2. 등록된 회원 이름 입력 (예: 홍길동)<br>3. 올바른 비밀번호 입력 (기본값: 1234)<br>4. 로그인 버튼 클릭 |
| **기대 결과** | - 메인 대시보드(`/`)로 리다이렉트<br>- 세션에 사용자 정보 저장<br>- 네비게이션에 사용자 이름 표시 |

### AUTH-002: 관리자 로그인

| 항목 | 내용 |
|------|------|

| **테스트 단계** | 1. `/auth/login` 페이지로 이동<br>2. 이름: "관리자" 입력<br>3. 비밀번호: "admin123" 입력<br>4. 로그인 버튼 클릭 |
